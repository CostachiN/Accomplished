Ce este web-designul?

Web Design-ul se refera la totalitatea etapelor de realizare a esteticii unui site web, cat si la utilizabilitatea acestuia.

Estetica site-ului web cuprinde mai multe aspecte: de la asezarea in pagina si schema de culori utilizata, pana la tot ce tine de User Experience (UX) si User Interface (UI).

Ce limbaje sunt folosite?

Pentru crearea unui site web sunt folosite doua limbaje:

  • HTML (Hypertext Markup Language) - este un limbaj de marcare. Se vor folosi taguri specifice pentru crearea unor elemente de design, precum asezarea in pagina
  • CSS (Cascading Style Sheets) - foloseste coduri pentru stilizarea paginilor web create cu ajutorul HTML

    • Cine este un web-designer?

    Persoana care detine toate cunostintele legate de realizarea esteticii unui site web se numeste Web Designer. Ce face acesta, mai exact?

    Un Web Designer profesionist va identifica obiectivele clientului pentru a putea aplica toate cunostintele sale in procesul de creare a site-ului web. Acesta va tine cont de interesele companiei pentru a crea imaginea vizuala, si pentru a organiza continutul site-ului.

    Care sunt elementele web-designului?

    Cele mai importante elemente ale Web Design-ului de calitate sunt urmatoarele:

  • Aspectul (Layout)
  • Imaginile
  • Schema de culori
  • Lizibitatea si Continutul
  • Tipografia
  • Navigarea
  • Adaptarea
